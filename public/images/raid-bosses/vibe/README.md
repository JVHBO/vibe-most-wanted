# VBMS (Vibe Most Wanted) Raid Boss Cards

Add the 5 VBMS boss card images here:

- `common.png` - Common rarity boss (1M HP)
- `rare.png` - Rare rarity boss (5M HP)
- `epic.png` - Epic rarity boss (25M HP)
- `legendary.png` - Legendary rarity boss (100M HP)
- `mythic.png` - Mythic rarity boss (500M HP)

## Note
These bosses will be dynamically fetched from JC's NFTs, but you can use placeholder images here for initial testing.

## Image Requirements
- Format: PNG
- Recommended size: 400x600px (2:3 aspect ratio)
- Should match the style of other NFT cards in the game
